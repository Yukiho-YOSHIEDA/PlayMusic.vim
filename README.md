# PlayMusic.vim

You can enjoy music with Vim

## Requirements

**`+sound`**

**libcanberra-dev**

### For Ubuntu

`sudo apt intsall libcanberra-dev`

## Usage

**Ubuntu19.04**

1. Prepare a `wav file` on `$HOME/Music`

2. Run `:PlayMusic`

**Windows**

1. Prepare a `wav file` or `mp3 file` on `$HOME/Music`

2. Run `:PlayMusic`
